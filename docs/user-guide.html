Qt GUI 使用者介面

系統採用 Qt 6.9.0 圖形化介面設計，提供完整的視窗化操作體驗：

🏦 ATM Banking System - Qt GUI 應用程式

主要視窗模組：
├── 🔑 loginWindow     - 登入視窗 (帳號/密碼輸入)
├── 👤 userWindow      - 用戶主介面 (功能選單)  
├── 👔 adminWindow     - 管理員介面 (管理功能)
├── 💰 depositWindow   - 存款視窗 (金額輸入)
├── 🏧 withdrowWindow  - 提款視窗 (快速金額按鈕)
├── 💸 transWindow     - 轉帳視窗 (目標帳戶選擇)
├── 🔐 changepasswordWindow - 密碼變更對話框
├── 📋 acclistWindow   - 帳戶列表表格
├── ➕ addaccWindow    - 新增帳戶表單
├── 🔄 setpasswordWindow - 密碼重設視窗
├── ⚙️ setstatusWindow - 帳戶狀態管理
├── 📊 summaryWindow   - 交易摘要彈出視窗
└── 📖 docuWindow      - 說明文檔視窗
操作方式： 透過滑鼠點擊按鈕、輸入框操作，支援QSS樣式美化介面

登入系統

Qt GUI 登入流程

  1. 啟動應用程式

    雙擊 ATM.exe 啟動Qt應用程式，將顯示登入視窗

  2. 輸入帳號

    在帳號輸入框中輸入您的使用者名稱（如：user）

  3. 輸入密碼

    在密碼輸入框中輸入對應的密碼（顯示為星號）

  4. 點擊登入按鈕

    點擊「登入」按鈕或按 Enter 鍵進行身份驗證

  5. 進入主介面

    驗證成功後將自動跳轉到用戶主視窗或管理員介面

安全提醒： 密碼錯誤過多次將觸發安全機制，系統將顯示錯誤對話框

Qt GUI 基本操作

用戶主視窗

成功登入後，將看到用戶主視窗，包含以下按鈕選項：

  • 💰 餘額查詢 - 顯示當前帳戶餘額
  • 💳 存款 - 開啟存款操作視窗
  • 🏧 提款 - 開啟提款操作視窗
  • 💸 轉帳 - 開啟轉帳功能視窗
  • 📊 交易記錄 - 查看最近5筆交易
  • 🔑 修改密碼 - 開啟密碼變更視窗
  • 🚪 登出 - 返回登入畫面

Qt 餘額查詢

在Qt GUI中查看帳戶餘額：

  1. 在用戶主視窗點擊「餘額查詢」按鈕
  2. 系統將在主視窗或彈出對話框中顯示當前餘額
  3. 餘額資訊即時從SQLite資料庫讀取
  4. 點擊「確定」或「返回」按鈕回到主視窗

Qt 提款操作

使用Qt GUI介面進行提款：

  1. 點擊主視窗的「提款」按鈕
  2. 在提款視窗中選擇提款方式：
    • 🎯 快速提款：點擊預設金額按鈕（1000/2000/3000/5000/10000）
    • ✏️ 自訂金額：在輸入框中輸入指定金額
  3. 點擊「確認提款」按鈕
  4. 系統驗證餘額並處理交易
  5. 顯示交易摘要視窗確認結果
Qt功能： 快速提款按鈕讓操作更便利，系統會即時檢查餘額是否足夠

Qt 存款操作

使用Qt介面進行存款：

  1. 點擊主視窗的「存款」按鈕
  2. 在存款視窗的金額輸入框中輸入存款金額
  3. 點擊「確認存款」按鈕
  4. 系統處理交易並更新SQLite資料庫
  5. 顯示交易摘要視窗確認存款完成
Qt優勢： 即時輸入驗證，防止無效金額輸入

Qt 轉帳功能

使用Qt介面進行帳戶間轉帳：

  1. 點擊主視窗的「轉帳」按鈕
  2. 在轉帳視窗中選擇或輸入目標帳戶
  3. 在金額輸入框中輸入轉帳金額
  4. 系統即時驗證目標帳戶是否存在
  5. 點擊「確認轉帳」按鈕
  6. 系統處理雙向帳戶餘額更新
  7. 顯示轉帳完成的交易摘要
Qt安全機制： 系統會在轉帳前驗證目標帳戶並確認餘額充足

Qt 交易記錄查詢

查看最近的交易記錄：

  1. 點擊主視窗的「交易記錄」按鈕
  2. 系統開啟交易記錄視窗
  3. 以表格形式顯示最近5筆交易
  4. 記錄包含：交易類型、金額、時間、餘額
  5. 點擊「返回」按鈕回到主視窗

Qt 密碼變更

修改帳戶密碼：

  1. 點擊主視窗的「修改密碼」按鈕
  2. 在密碼變更視窗中輸入舊密碼
  3. 輸入新密碼（兩次確認）
  4. 點擊「確認變更」按鈕
  5. 系統使用SHA-256加密新密碼
  6. 更新SQLite資料庫中的密碼雜湊
安全保護： 所有密碼都經過SHA-256雜湊加密存儲

帳戶管理

查看交易紀錄

檢視帳戶的交易歷史：

  1. 選擇 "5. 交易紀錄"
  2. 選擇查詢期間（最近 10 筆/全部）
  3. 查看詳細交易資訊

Qt 密碼修改

在Qt介面中更改帳戶密碼以提高安全性：

  1. 點擊用戶主視窗的「修改密碼」按鈕
  2. 在密碼變更視窗中輸入當前密碼
  3. 輸入新密碼（兩次確認）
  4. 點擊「確認變更」按鈕
  5. 系統更新密碼並使用SHA-256加密儲存
  6. 顯示密碼更新成功的確認對話框
Qt安全機制： 系統會驗證舊密碼正確性才允許變更

Qt 管理員功能

管理員登入

使用管理員權限登入Qt系統：

  1. 在登入視窗輸入管理員帳號：admin
  2. 輸入管理員密碼：admin123
  3. 點擊「登入」按鈕
  4. 系統驗證後將進入管理員主視窗
管理員介面： 具備完整的帳戶管理與系統控制功能

管理員主視窗

管理員主視窗提供以下功能按鈕：

  • 📋 帳戶列表 - 查看所有用戶帳戶資訊
  • ➕ 新增帳戶 - 創建新的用戶或管理員帳戶
  • 🔑 重設密碼 - 強制重設任何帳戶密碼
  • ⚙️ 帳戶狀態 - 啟用/停用/鎖定帳戶
  • 📖 說明文檔 - 查看系統使用說明
  • 🚪 登出 - 返回登入畫面

Qt 帳戶列表管理

使用Qt表格視窗管理所有帳戶：

  1. 點擊管理員主視窗的「帳戶列表」按鈕
  2. 系統開啟帳戶列表視窗，顯示完整的帳戶表格
  3. 表格包含欄位：ID、帳號、狀態、餘額、角色
  4. 可以瀏覽所有用戶和管理員帳戶資訊
  5. 點擊「返回」按鈕回到管理員主視窗

Qt 新增帳戶

使用Qt對話框創建新帳戶：

  1. 點擊管理員主視窗的「新增帳戶」按鈕
  2. 在新增帳戶視窗中填寫：
    • 🆔 帳號名稱：輸入新用戶的帳號
    • 🔐 初始密碼：設定預設密碼
    • 💰 初始餘額：設定帳戶起始金額
    • 👤 帳戶角色：選擇用戶或管理員權限
  3. 點擊「確認新增」按鈕
  4. 系統將新帳戶資料寫入SQLite資料庫
  5. 顯示新增成功的確認訊息

Qt 密碼重設

管理員強制重設任何帳戶密碼：

  1. 點擊管理員主視窗的「重設密碼」按鈕
  2. 在密碼重設視窗中選擇目標帳戶
  3. 輸入新密碼（兩次確認）
  4. 點擊「確認重設」按鈕
  5. 系統使用SHA-256加密新密碼並更新資料庫
  6. 顯示重設完成的通知
管理員權限： 此功能僅限管理員使用，可重設任何帳戶密碼

Qt 帳戶狀態管理

控制帳戶的啟用/停用狀態：

  1. 點擊管理員主視窗的「帳戶狀態」按鈕
  2. 在狀態管理視窗中選擇目標帳戶
  3. 從下拉選單選擇新狀態：
    • ✅ 啟用：帳戶可正常使用
    • ⛔ 停用：暫時停用帳戶
    • 🔒 鎖定：完全鎖定帳戶
  4. 點擊「確認變更」按鈕
  5. 系統立即更新帳戶狀態並同步到資料庫
安全控制： 停用的帳戶無法登入系統，確保帳戶安全

系統監控

監控系統狀態：

  • 交易統計： 查看系統交易統計資料
  • 用戶活動： 監控用戶活動記錄
  • 系統日誌： 查看系統運行日誌

Qt系統安全提醒

保護登入密碼

在Qt登入視窗輸入密碼時注意周圍環境，密碼以星號顯示保護隱私

90秒會話超時

系統設有90秒自動登出機制，長時間未操作將自動返回登入視窗

定期更換密碼

使用Qt密碼變更視窗定期更新密碼，系統採用SHA-256加密保護

異常交易提醒

發現異常交易請立即使用管理員功能檢查帳戶狀態或聯繫系統管理員

資料庫安全

SQLite資料庫檔案(atm.db)包含敏感資訊，請妥善保管避免外洩

正確關閉應用程式

使用Qt視窗的登出按鈕或關閉按鈕正確退出，避免強制終止程式

Qt系統故障排除

Q: Qt應用程式無法啟動怎麼辦？

A: 確認系統已安裝Qt 6.9.0運行庫，檢查atm.db資料庫檔案是否存在於應用程式目錄中。

Q: 忘記登入密碼怎麼辦？

A: 請聯繫管理員使用「重設密碼」功能重置密碼，或使用管理員帳號(admin/admin123)進行重設。

Q: 帳戶狀態顯示為停用如何處理？

A: 請聯繫管理員使用「帳戶狀態」管理功能將帳戶重新啟用。

Q: Qt視窗無法正常顯示怎麼辦？

A: 檢查螢幕解析度設定，確認QSS樣式檔案完整，重新啟動應用程式。

Q: 轉帳操作失敗的原因？

A: 可能原因包括：餘額不足、目標帳戶不存在、SQLite資料庫連線錯誤、輸入金額格式無效。

Q: 如何查看SQLite資料庫中的完整記錄？

A: 系統僅顯示最近5筆交易記錄，完整資料可使用SQLite管理工具查看atm.db檔案。
