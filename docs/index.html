🏦 ATM 自動櫃員機系統

現代化的Qt GUI銀行自動櫃員機系統，採用Controller模式設計與SQLite資料庫整合

企業級自動櫃員機模擬系統，運用Qt 6.9.0框架開發的完整GUI金融應用程式， 具備14個視窗模組、SQLite資料庫整合、Signal-Slot通訊機制以及完整的Controller架構設計。

核心功能特色

完整的銀行業務功能與企業級安全機制

用戶功能模組

  • 🔐 Qt GUI安全登入介面 (使用者身份驗證)
  • 💰 餘額查詢視窗 (即時SQLite查詢)
  • 💳 存款功能 (金額輸入框+確認按鈕)
  • 🏧 提款服務 (快速金額按鈕: 1000/2000/3000/5000/10000)
  • 💸 轉帳視窗 (目標帳戶選擇+金額確認)
  • 📊 交易歷史表格 (最近5筆記錄顯示)
  • 🔑 密碼變更對話框 (舊密碼驗證)
  • 📋 交易摘要彈出視窗 (操作完成反饋)

管理員功能模組

  • 🛡️ 管理員專用Qt登入視窗
  • 📋 帳戶列表表格視窗 (ID/帳號/狀態/餘額顯示)
  • ➕ 新增帳戶對話框 (使用者/管理員角色選擇)
  • 🔄 密碼重設視窗 (帳戶選擇+新密碼輸入)
  • ⚙️ 帳戶狀態管理介面 (啟用/停用下拉選單)
  • 📊 SQLite資料庫管理 (Controller統一操作)
  • 🔍 帳戶查詢與驗證 (即時資料更新)
  • 🎛️ 批量管理操作界面

Qt架構與安全

  • ⏰ 90秒會話超時 + 自動返回登入視窗
  • 🔐 SQLite SHA-256密碼雜湊儲存
  • 🚫 Qt輸入驗證與錯誤對話框
  • 💾 SQLite事務完整性保護
  • 📋 交易記錄持久化 (5筆歷史)
  • 🔒 Controller靜態狀態管理
  • ⚡ 即時SQLite餘額驗證
  • 🎯 Qt Signal-Slot線程安全通訊
